# GymFlow

אפליקציית חדר כושר בעברית לניהול תוכנית אימונים, תרגילים, סטים, חזרות ומשקל לכל אימון.

## פתיחה מקומית

פתח את `index.html` בדפדפן.

## פרסום לטלפון עם GitHub Pages

1. צור ריפו חדש ב-GitHub.
2. העלה אליו את כל הקבצים שבתיקייה הזו.
3. ב-GitHub היכנס ל-`Settings` ואז `Pages`.
4. תחת `Build and deployment` בחר:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` ו-`/ (root)`
5. שמור והמתן כדקה-שתיים.
6. GitHub יציג לך קישור ציבורי שאפשר לפתוח מהטלפון.

## שמירה למסך הבית בטלפון

לאחר שהאתר באוויר:

- באייפון: פתח ב-Safari, לחץ `Share`, ואז `Add to Home Screen`.
- באנדרואיד: פתח ב-Chrome, לחץ על התפריט, ואז `Add to Home screen` או `Install app`.

## חשוב לדעת

- הנתונים נשמרים בתוך הדפדפן של אותו מכשיר באמצעות `localStorage`.
- אם תפתח את האתר בטלפון ובמחשב, לכל מכשיר תהיה שמירה נפרדת.
